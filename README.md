## 📁 目录说明

| Path | 用途 |
| --- | --- |
| `profile.md` | 我的学员画像、学习起点、目标方向 |
| `learning-plan.md` | 8 周学习计划与每周里程碑 |
| `daily/` | 每日 daily note（`YYYY-MM-DD.md`），记录今日学习 + 打卡草稿 |
| `tasks/` | WCB 课程任务的拆解笔记（按任务编号或主题命名） |
| `experiments/` | Vibe Coding / MCP / 钱包 / 合约小实验 |
| `handbook-feedback/` | 对 Handbook 的反馈：错别字、概念不清、资料过期、结构建议 |
| `hackathon/` | Hackathon 想法、调研、原型设计 |
| `submissions/` | 提交给 WCB 平台的作品备份（写完后同步提交到 WCB） |
| `templates/` | daily note / task note 模板 |

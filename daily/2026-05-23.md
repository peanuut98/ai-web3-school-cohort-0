## 📖 Handbook 复习记录（Day 1–Day 3 串线）

把已经学过的三节自己复述一遍，没复述出来的就是漏掉的：

- **Network（Day 1）**：区块 = 一批交易打包后的单位；共识 = 节点之间对“链长什么样”达成一致；L2 = 在主网安全性之上、成本更低的一层；RPC = 我和节点说话用的接口。
- **Cryptography（Day 2）**：私钥签名 → 公钥验证 → 地址是公钥 hash 的一段。Hash 的雪崩效应：1bit 输入差异 → 整个输出完全不同（Day 2 用 hello / Hello 实测过）。
- **Wallet（Day 3，部分）**：钱包 ≠ 装币的口袋，而是**私钥管理工具 + 签名入口**。助记词 → 私钥 → 公钥 → 地址 是一条单向派生链路。MetaMask 是这条链路的 UI。

补 Day 3 没填完的概念（今天补的）：

- 助记词（Seed Phrase）：12 / 24 个英文单词，是私钥的人类可读版本。背后是 BIP39 标准。
- 派生路径（Derivation Path）：像 `m/44'/60'/0'/0/0` 这种结构，决定了从同一组助记词派生出哪一个具体地址。`60'` 是 EVM 链的标识。
- 热钱包：联网设备上的钱包（MetaMask 这种），方便但攻击面大。
- 冷钱包：从不联网的设备（硬件钱包 / 纸钱包），安全但操作不便。
